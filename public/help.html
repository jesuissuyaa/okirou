OUJバーチャル自習室について

OUJバーチャル自習室は、以下のページで構成されています。
  1. 自習室
  2. 休憩室
  3. マイページ
  4. 使い方ガイド
1. 自習室について
  • お互いの勉強風景をリアルタイムで見れます
  • 勉強する内容に応じた自習室を選べます (所属コース/プログラム以外の自習室に入ってもokです)
  • 音声は流れません
  • 自分の映像を流したくない場合は、かわりに自分のプロフィール画像を表示することもできます
2. 休憩室について
  • 音声通話とテキストチャットができます
  • どの休憩室に入るかは選べません
  • マイクがない場合は、テキストチャットのみの休憩室を利用できます
  • 「自習室に戻る」ボタンで、元々いた自習室の自分の席に戻れます
3.マイページについて
  • ユーザー名、プロフィール画像が変更できます
  • パスワードの変更もできます
4. 使い方ガイドについて
このページです。わからないことがあるときは、このページをお読みください。
他のページからは、右上のからこのページに飛べます。
使い方ガイドに書いていないことについて聞きたい場合は、問合せ先にご連絡ください。

新規登録の流れ

1. 新規登録ページに行く
2. 利用条件を読み、新規登録に進む
3. 必要事項を入力し、「新規登録」ボタンを押す
4. 放送大学Gmailに行く
5. メールを開く (件名: 「OUJバーチャル自習室のメールアドレスの確認」)
6. メール内のリンクをクリックする
7. トップページからログインする
8. 完了！

放送大学Gmailのメールアドレスを確認する

1. システムWakabaにログインする
https://www.wakaba.ouj.ac.jp
2. トップページの「Gmail」をクリック
3. Gmailのメールアドレスを確認する
4. 完了！
新規登録に進む場合は、メールアドレスをコピーして、
新規登録ページの「メールアドレス」欄に貼り付けてください

自習室でカメラが利用できない / 休憩室でマイクが利用できない

自習室/休憩室で「※カメラ/マイクが使用できません。」と表示された場合、次の原因が考えられます。
  1. ブラウザでカメラ/マイクのアクセスを許可していない
  2. 他のアプリケーション(SkypeやLineなど)でカメラ/マイクを使用している
  3. その他
1. ブラウザでカメラ/マイクのアクセスを許可していない場合
(休憩室の場合は、画像の中のカメラがマイクになります)
ページを更新する
2. 他のアプリケーション(SkypeやLineなど)でカメラ/マイクを使用している場合
カメラ/マイクを使用しているアプリケーションを閉じる → ページを更新する
※何回か更新してもエラーが出る場合は、タスクマネージャーからアプリケーションを閉じるか、PCを再起動してください。
3. その他
カメラ/マイクとPCが接続されていないなど、PC側の原因が考えられます。
  • カメラ/マイクがUSB接続の場合、USBを一度抜いてもう一度挿す
  • カメラ/マイクのデバイスドライバを最新版に更新する
などの対策が考えられます。
対策をとってもエラーが出つづける場合は、
  • 自習室: 「PCにカメラがついていないとき」をお読みください
  • 休憩室: マイク不要のロビーをご利用ください
    (ロビーへは、休憩室でマイクエラーが出たときのボタンから行けます)

PCにカメラがついていないとき

1. デスクトップキャプチャーをインストールする
ニコ生デスクトップキャプチャー(XP)などがあります。
※デスクトップキャプチャーがWebカメラとしてブラウザに検知されるはずですが、
    ニコ生デスクトップキャプチャー以外のソフトウェアの動作は確認しておりません。
2. 自習室で「カメラを使用しない」を選択
※画質設定の都合上、画面共有はできません。

パスワードを変更する

1. アカウント設定に行く
2. 「パスワードを再設定」ボタンをクリック
3. 再設定用メールが届くので、ここの手順でメール内のリンクをクリック
4. 新しいパスワードを設定する
5. 新しいパスワードでログインする
6. 完了！
